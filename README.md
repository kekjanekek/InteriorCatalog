# InteriorCatalog
Описание проекта

InteriorCatalog — WPF приложение для управления каталогами мебели.
Позволяет просматривать, сортировать, фильтровать и редактировать предметы мебели из разных каталогов, а также просматривать изображения товаров.

Проект разделён на:

основное WPF-приложение (InteriorCatalog)
библиотеку бизнес-логики (Model)
Основной функционал
Работа с каталогами
В главном меню пользователь выбирает каталог из выпадающего списка
Кнопка “Показать каталог” активируется только после выбора каталога
После нажатия открывается окно с таблицей товаров выбранного каталога
Окно каталога (CatalogWindow)

В окне отображается таблица мебели:

Имя
Артикул
Модель
Бренд
Цена
Кнопка открытия изображения
Фильтрация
ComboBox “Тип мебели”

Позволяет фильтровать товары по типу:

All — показать все товары
Chair
Sofa
Table
Bed
Stool
Armchair

При выборе типа:

таблица обновляется автоматически
отображаются только элементы выбранного типа
Сортировка

Кнопки сортировки:

По артикулу
сортировка по возрастанию
сортировка по убыванию
По названию
A → Z
Z → A
По цене
по возрастанию
по убыванию
Сгруппировать
выполняет сортировку PrioritySort
группирует товары по бренду, модели, названию и артикулу
Просмотр изображения товара

В таблице есть кнопка “Открыть”:

При нажатии:

открывается новое окно FurnitureImageWindow
отображается изображение выбранного предмета
показывается название товара
Удаление и восстановление
Удаление товара
удаляет выбранный элемент по артикулу
обновляет таблицу
сохраняет удалённый объект
Undo (вернуть товар)
возвращает последний удалённый предмет
кнопка активируется только после удаления
отключается после восстановления
Фильтр + сортировка + удаление

Все операции работают совместно:

фильтр не сбрасывается при сортировке
сортировка применяется к текущему отображению
удаление обновляет текущий список
Архитектура проекта
Model (бизнес-логика)

Содержит:

Furniture (абстрактный класс)
наследники:
Chair
Table
Sofa
Bed
Armchair
Stool
Особенности:
каждый класс переопределяет Price
цена зависит от дополнительных параметров конкретного типа
Интерфейсы
IFurnitureCatalog

Отвечает за управление каталогом:

Add(Furniture item)
Add(Furniture[] items)
Remove(string article)
Remove(Type type)
ISortable

Реализован в FurnitureCatalog:

Sort(bool ascending)
SortByName(bool ascending)
SortByPrice(bool ascending)
FurnitureCatalog (partial class)

Содержит:

массив товаров
операции добавления/удаления
сортировки
PrioritySort (группировка товаров)
Сериализация

Поддерживается работа с файлами:

JSON (основной формат)
XML (дополнительный формат)

Реализована через:

абстрактный класс сериализации
наследники для JSON и XML

Данные:

сохраняются при первом запуске
могут переноситься между форматами
UI (WPF)

В интерфейсе используются:

DataGrid (таблица товаров)
ComboBox (фильтр и выбор каталога)
Buttons (сортировка, удаление, undo, открытие изображения)
TextBlock (отображение общей стоимости)
Подсчёт стоимости

Внизу окна отображается суммарная стоимость всех отображаемых товаров.

Особенности реализации
массивы вместо коллекций
ручная реализация фильтрации
ручная реализация сортировок (bubble sort)
использование делегатов (Action<string>, Func<T,bool>)
operator overload для цены
partial class для расширения каталога
WPF взаимодействие через code-behind
Открытие изображений

Каждый предмет мебели содержит путь к изображению.

Изображения:

загружаются из локальной папки
отображаются в отдельном окне
Логика интерфейса
фильтр влияет на отображаемый список
сортировка применяется к текущему виду
удаление влияет на базовый каталог
undo работает только для последнего удаления
